## 📦 インストール（3分で完了）

> ⚠️ **インターネット接続が必要です**（初回のAIダウンロードのため）  
> インストール後は **オフラインでも使えます**

### 必要なもの

- Windows 10 / 11
- マイク（内蔵でもヘッドセットでもOK）
- インターネット接続（初回のセットアップ時のみ）

### 手順

```
① このページの緑の「Code」→「Download ZIP」をクリック
② ZIPを解凍して、好きな場所に置く
③ フォルダ内の「1_Setup.bat」をダブルクリック（自動でPythonなどの準備が始まります）
④ 完了したら、アプリが起動！

**🎉 これで準備完了です。次回からはWindowsを立ち上げるだけで自動的に使えるようになります。**
（手動で起動したい場合は `2_Start.bat` を使ってください）

## 🎤 使い方

### 基本操作

```
1. テキストを入力したい場所をクリック（メモ帳、メール、チャットなど）
2. 右Ctrlキーを押したまま、マイクに向かって話す
3. キーを離す → 話した内容が自動で入力される！
```

> 💡 **コツ**: キーを押してから少し間を置いて話し始めると、認識精度が上がります

### カーソルで状態がわかります

| カーソルの形 | 意味 |
|---|---|
| ＋（十字） | 🎤 録音中（聞いています） |
| ↗〇（砂時計） | ⏳ AIが文字に変換中 |
| 通常のカーソル | ✅ 完了！ |

### 初回起動時

初回は **設定画面** が自動で開きます：
- 📌 使うマイクを選ぶ
- 📌 トリガーキーを確認（デフォルト: 右Ctrl）
- 📌 AIモデルは PC のスペックに合わせて自動で選ばれています
- 📌 「この設定で始める」を押すと、AIのダウンロードが始まります（数分）

## ⚙ 設定を変えたいとき

アプリ画面の **「⚙ 設定」ボタン** を押すと設定画面が開きます。  
（音声入力を停止してから開いてください）

| 設定 | 説明 |
|---|---|
| **マイク** | 使うマイクを選べます |
| **トリガーキー** | 右Ctrl 以外のキーにも変更可能 |
| **AIモデル** | 精度と速度のバランスを選択（PCのRAMに合わせて自動推薦） |
| **自動Enter** | 入力後に自動でEnterキーを押します |
| **ノイズ除去** | AIの聞き間違い（ゴミ文字）を自動カット |
| **録音ゲイン** | マイクの音量が小さいときに上げてみてください |

## ❓ 困ったとき

| 症状 | 対処法 |
|---|---|
| キーを押しても反応しない | `2_Start.bat` から起動していますか？管理者権限が必要です |
| 特定のアプリで入力されない | アプリとの相性の問題です。メモ帳で動くか確認してください |
| キーが押しっぱなしになった | アプリ画面の **「⌨ キーリセット」** を押してください |
| ダウンロード中に止まった | ネット接続を確認し、再起動すれば続きから再開します |

## 🔧 開発者向け情報

> ここから先はエンジニア・コントリビューター向けの技術情報です。

### アーキテクチャ

```
WhisperMoji
├── main.py              ← エントリーポイント & UI統合
├── core/                ← 基盤レイヤー
│   ├── i18n.py          ← OS言語自動検知 → lang/*.json 読み込み
│   └── config.py        ← config.json 永続化、RAM検知、モデル推薦
├── engine/              ← 音声処理レイヤー
│   ├── audio.py         ← sounddevice ラッパー（マイク列挙・バッファ管理）
│   ├── whisper_engine.py← faster-whisper 推論（VAD + ノイズフィルタ）
│   └── keyboard_input.py← クリップボード経由テキスト挿入
├── ui/                  ← プレゼンテーションレイヤー
│   ├── constants.py     ← カラーパレット・フォント定義
│   ├── cursor_manager.py← Win32 API カーソル差し替え
│   └── settings_window.py← customtkinter 設定ダイアログ
└── lang/                ← i18n リソース
    ├── ja.json / en.json / ko.json / zh.json
```

### 技術スタック

| 項目 | 詳細 |
|---|---|
| 音声認識 | [faster-whisper](https://github.com/guillaumekln/faster-whisper)（CTranslate2 / int8量子化） |
| 実行環境 | CPU専用（GPU不使用） |
| UI | customtkinter（ダークモード固定） |
| ホットキー | `keyboard`（`WH_KEYBOARD_LL` フック / 要管理者権限） |
| テキスト入力 | クリップボード経由 `Ctrl+V` シミュレーション |
| カーソル表示 | Win32 API（`SetSystemCursor` / `SystemParametersInfo`） |
| 多重起動防止 | Windows Mutex |
| 自動更新 | PyPI API ポーリング → `pip install --upgrade` |

### 管理者権限が必要な理由

`keyboard` ライブラリは `WH_KEYBOARD_LL` フックを使用しています。管理者権限なしでは VS Code 等の一部アプリからキーイベントを取得できません。`1_Setup.bat` でタスクスケジューラに最上位権限で登録し、以降は UAC なしで管理者起動します。

### ビルド & 実行

```bash
# 依存インストール
pip install -r requirements.txt

# 起動（管理者権限の PowerShell / cmd で実行）
python main.py
```
