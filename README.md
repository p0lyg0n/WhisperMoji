## 📦 インストール（3分で完了）

> ⚠️ 初回のみ、AIモデルをダウンロードするためにインターネット接続が必要です。

### 必要なもの

- Windows 10 / 11
- マイク（内蔵でもヘッドセットでもOK）
- インターネット接続（初回のセットアップ時のみ）

### 手順

```
① このページの緑の「Code」→「Download ZIP」をクリック
② ZIPを解凍して、好きな場所に置く
③ フォルダ内の「1_Setup.bat」をダブルクリック（自動でPythonなどの準備が始まります）
④ 完了したら、アプリが起動！
```

**🎉 これで準備完了です。次回からはWindowsを立ち上げるだけで自動的に使えるようになります。**  
（手動で起動したい場合は `2_Start.bat` を使ってください）

## 🎤 使い方

### 基本操作

```
1. 文字を入れたい場所をクリック
2. 右Ctrlキーを押したまま、話す
3. キーを離すと、文字が入力される！
```

> 💡 **コツ**: キーを押してから一呼吸置いてから話し出すと、より正確に認識されます。

### カーソルで状態が確認できます

| カーソルの形 | 意味 |
|---|---|
| ＋（十字） | 🎤 録音中 |
| ⏳（砂時計） | ⏳ AIが文字にしています |
| 通常のカーソル | ✅ 完了！ |

## ⚙ 設定を変えたいとき

アプリ画面の **「⚙ 設定」** ボタンから変更できます。

| 設定 | 説明 |
|---|---|
| **マイク** | 使うマイクを選べます |
| **トリガーキー** | キーを変更できます（デフォルト: 右Ctrl） |
| **AIモデル** | PCの性能に合わせて自動で選ばれます |
| **自動Enter** | 入力した後に改行（Enter）を入れます |
| **録音ゲイン** | 声が小さいときに調整してください |

---

## ❓ 困ったとき

- **キーを押しても反応しない**: `2_Start.bat` から起動し直してみてください。
- **特定のアプリで入らない**: 一部のアプリとの相性で入力できない場合があります。
- **キーが押しっぱなしになった**: アプリ画面の **「⌨ キーリセット」** を押してください。
